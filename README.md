# Webregator
A webapp that generates and saves all h tags and a tags from a website to the database. Currently, there are two frontend versions: AngularJS and React.

## Installation
 - rake bower:install
 - yarn install

## Serve app
 - foreman start

## Versions
 - On root: Angular version
 - React: localhost:5555/react

## Testing
### Backend testing
rails test test/*
